# v2.1.3 更新日志

发布日期：2026-02-27

## 🐛 修复

- 修复智能缩放/智能气泡模式下，竖排 `has_br=False` 场景的断句失效问题：竖排布局预处理改为先规范化 `[BR]/<br>/【BR】` 为换行，再注入 `<H>` 标签，避免断句标记被标签化后失效并错误收敛为单行。
- 修复模型辅助合并在 `other` 包裹场景下无法合并的问题：特殊合并阶段允许 `other` 参与包裹关系建图作为桥接框，使 `changfangtiao` / `balloon` / `qipao` 与无标签框在满足“完全包裹”关系时可正确归组；`other` 仍保持辅助角色，不参与最终文本块合并结果。
